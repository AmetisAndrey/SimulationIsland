# 🏝️ Island Simulation

## 🧠 Описание проекта
**Island Simulation** — это объектно-ориентированная многопоточная симуляция экосистемы острова.  
Мир представлен в виде двумерной сетки, где обитают растения 🌿,  
травоядные 🐇🐎🐐 и хищники 🐺🐻🦊.  

Каждый вид взаимодействует с другими по заданным правилам:
- 🌿 растения растут со временем;  
- 🐇 травоядные едят растения и размножаются при наличии пары;  
- 🐺 хищники охотятся на травоядных с вероятностью успеха;  
- ⚰️ животные умирают от голода или старости;  
- ♻️ при вымирании вида срабатывает респаун (возрождение).  

Процесс полностью **автоматизирован**, визуализируется с помощью **Swing GUI**,  
и управляется таймерами и потоками (`ScheduledExecutorService`).  

---

## ⚙️ Основные возможности
✅ Объектно-ориентированная иерархия животных (абстрактный класс `Animal`)  
✅ Реализация поведения — еда, движение, размножение, смерть  
✅ Многопоточность: параллельный жизненный цикл и рост растений  
✅ Псевдографика на Swing с использованием юникод-эмоджи 🐺🐇🐻🌿  
✅ Настраиваемые параметры через класс `Config`  
✅ Автоматическая статистика по видам (в консоли и GUI)  
✅ Защита от перенаселения (лимиты в клетках и по острову)  
✅ Респаун вымерших видов ♻️  
✅ Случайная генерация острова и начальной популяции

## Архитектура проекта
src/
├── Main.java # Точка входа (создает симуляцию и UI)
│
├── island/
│ ├── Island.java # Модель острова и сетка клеток
│ ├── Cell.java # Клетка: растения и животные
│ ├── Config.java # Параметры симуляции (веса, лимиты, скорость и т.д.)
│ ├── Simulation.java # Основной движок симуляции (потоки, расписания)
│
├── animals/
│ ├── Animal.java # Абстрактный класс: общее поведение
│ ├── Herbivore.java # Травоядные (ест растения)
│ ├── Predator.java # Хищники (ест других животных)
│ ├── ... (Wolf, Bear, Deer, Rabbit, Goat и др.)
│
├── plant/
│ └── Plant.java # Класс растения (рост, масса)
│
├── utils/
│ ├── RandomGenerator.java # Потокобезопасный рандом
│ └── AnimalFactory.java # Фабрика для создания экземпляров животных
│
└── ui/
└── IslandFrame.java # Swing-интерфейс (визуализация острова)
